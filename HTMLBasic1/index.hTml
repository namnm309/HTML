1.2 Cấu trúc tài liệu HTML Cơ bản

Tìm hiểu về cấu trúc cơ bản của một tài liệu HTML

<meta> <title> <head>

    Thẻ meta: chứa thông tin metadata của trang
    Thẻ title: tiêu đề trang (hiển thị trên tab browser)
    Thẻ head: chứa thông tin header của trang
    Thẻ body: chứa nội dung chính của trang
Cấu trúc HTML cơ bản

Code : 
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Mô tả trang web">
    <title>Tiêu đề trang web</title>
</head>
<body>
    <!-- Nội dung trang web -->
</body>
</html>

2.1 Danh sách Trung cấp

Tạo danh sách có thứ tự và không có thứ tự với ul, ol, li

<ul></ul> <ol></ol> <li></li>

1.7 Hình ảnh Cơ bản

Hiển thị hình ảnh trong HTML với thẻ img

<img>

    Thẻ img dùng để hiển thị hình ảnh trong HTML
    Thuộc tính src: đường dẫn đến hình ảnh
    Thuộc tính alt: văn bản thay thế khi không tải được ảnh
    Thuộc tính width và height: điều chỉnh kích thước ảnh
Hình ảnh với kích thước vừa phải



Hình ảnh responsive (tự động điều chỉnh)



Code : 
<!-- Hình ảnh với kích thước cố định -->
<img src="/image/1.png" alt="Ảnh minh họa" width="300" height="200">

<!-- Hình ảnh responsive -->
<img src="/image/2.png" alt="Ảnh responsive" style="max-width: 100%; height: auto;">

3.1 HTML5 Semantic Elements Nâng cao

Các thẻ semantic giúp cấu trúc trang web rõ ràng và có ý nghĩa

<header> <nav> <main> <section>

    Thẻ meta: chứa thông tin metadata của trang
    Thẻ title: tiêu đề trang (hiển thị trên tab browser)
    Thẻ head: chứa thông tin header của trang
    Thẻ body: chứa nội dung chính của trang
Cấu trúc HTML cơ bản

Code : 
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Mô tả trang web">
    <title>Tiêu đề trang web</title>
</head>
<body>
    <!-- Nội dung trang web -->
</body>
</html>
<header> <nav> <main> <section>

    HTML5 Semantic Elements - Giúp cấu trúc trang web rõ ràng hơn:
    Thẻ header: phần đầu trang (tiêu đề, logo, menu)
    Thẻ nav: điều hướng (menu chính)
    Thẻ main: nội dung chính của trang
    Thẻ section: phần/mục trong trang
    Thẻ article: bài viết độc lập
    Thẻ aside: thanh bên (sidebar)
    Thẻ footer: phần chân trang

3.8 Text Formatting Nâng cao

Các thẻ định dạng văn bản nâng cao như kbd, samp, var, sub, sup

<kbd> <samp> <var> <sub> <sup>

    Thẻ kbd: keyboard input (phím bấm)
    Thẻ samp: sample output (kết quả mẫu)
    Thẻ var: variable (biến số)
    Thẻ sub: subscript (chỉ số dưới)
    Thẻ sup: superscript (chỉ số trên)
    Thẻ small: văn bản nhỏ
Text Formatting Elements

Để copy text, bấm Ctrl + C

Computer output: File saved successfully!

Trong công thức này, x là biến số

Công thức hóa học: H2O

Công thức toán: E = mc2

Đây là văn bản nhỏ hơn bình thường

Code : 
<p>Bấm <kbd>Ctrl</kbd> + <kbd>C</kbd> để copy</p>
<p>Output: <samp>File saved!</samp></p>
<p>Biến <var>x</var> trong công thức</p>
<p>H<sub>2</sub>O và E = mc<sup>2</sup></p>
<p><small>Văn bản nhỏ</small></p>

3.9 Văn bản đặc biệt Nâng cao

Các thẻ văn bản đặc biệt như wbr, bdi, bdo, ruby

<wbr> <bdi> <bdo>

    Thẻ wbr: word break opportunity (cơ hội ngắt từ)
    Thẻ bdi: bidirectional isolate (cô lập văn bản hai chiều)
    Thẻ bdo: bidirectional override (ghi đè hướng văn bản)
    Thẻ ruby, rt, rp: chú thích đọc (dùng cho tiếng Nhật/Trung)
